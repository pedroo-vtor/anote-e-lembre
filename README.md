# Anote e Lembre - App de Gerenciamento de Tarefas

# AV2 – Código de Alta Performance - Mobile e Maturidade em Desenvolvimento de Software

## Descrição Geral do Projeto

O *Anote e Lembre* é um aplicativo móvel minimalista para **gerenciamento de tarefas**, projetado para proporcionar uma curva de aprendizado extremamente suave. O sistema foi **desenvolvido em Flutter**, seguindo o **padrão arquitetural MVVM** (Model-View-ViewModel) com o **uso do Provider** para gerenciamento de estado. O aplicativo **integra Firebase Auth** para autenticação segura via e-mail e senha, **e Cloud Firestore** para armazenamento persistentes de dados em tempo real em um banco de dados NoSQL.
